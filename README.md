# Sum Product Loop

Консольное приложение на C# для вычисления суммы и произведения чисел от 1 до n.

## Функциональность

- Ввод целого числа n
- Вычисление суммы чисел от 1 до n
- Вычисление произведения чисел от 1 до n
- Вывод на каждом шаге цикла

## Пример работы

Введите целое число (от 3 до 9)
5
шаг = 1 сумма = 1 произведение = 1
шаг = 2 сумма = 3 произведение = 2
шаг = 3 сумма = 6 произведение = 6
шаг = 4 сумма = 10 произведение = 24
шаг = 5 сумма = 15 произведение = 120

## Куда можно встроить

- В обучающие программы
- В демонстрацию циклов
- В примеры накопления сумм

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Для больших n рекомендуется использовать long или decimal.
